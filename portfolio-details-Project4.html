Parking Ticket Management System

Ticket management is based on the laws of Ontario and supports either opted-out or opted in to Bills 25 & 47. Administrative Monetary Penalty enforcement of parking and licensing (taxi, lottery, massage parlour, retail etc) infractions is also supported. The system is developed and supported in Ontario, Canada with a 20+ year history of satisfied clients.

The data is searched via Ticket Tracer’s Select windows that resemble an Excel spreadsheet. The search criteria are entered.

The user is looking for all of the tickets issued in March 2019 that are currently at ‘event’ CRPD Paid. Tickets have a life and go through various events. Rows of tickets meeting the criteria are returned:



This type of ‘spreadsheet’ search is consistent across Ticket Tracer. The same techniques can be used to search for tickets, infraction code reference data, officer reference data etc. To see the detail for any row, the user just double clicks on it.

The ticket detail for that PIN or ticket is displayed. Multiple tickets and windows can be open at the same time for a number of different queries or windows.
